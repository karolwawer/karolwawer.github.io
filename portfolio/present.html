Moje umiejętności

11 czerwca 2018 roku zakończyłem kurs programowania w Akademii 108 w trybie dziennego bootcampu. Wiedza jaką zdobyłem dzięki temu kursowi i ludzie jakich poznałem w jego trakcie, utwierdziły mnie w przekonaniu, że praca jako Front End Developer jest dla mnie celem zawodowym numer 1

×

Technologie

Tworząc strony internetowe wykorzystuję technologie jakich nauczyłem się na kursie programowania w Akademii 108. Wiem, że praktyka czini mistrza, więc staram się rozwijać i zdobywać nową wiedze z poznanych mi technologii.

W projektach wykorzystuję nowe znaczniki semantyczne w HTML5. Potrafię wykorzystywać HTML z CSS do tworzenia responsywnych stron i animacji.

HTML5

Hypertext Markup Language

Znam podstawowe oraz zaawansowane właściwości, pseudoelementy, pseudoklasy, selektory i pseudoselektory. Wykorzystuję flex-box'a oraz potrafię tworzyć animacje.

CSS3

Cascading Style Sheets

Rozumiem programowanie obiektowe, działanie obiektowego modelu domu oraz AJAX'a i JSON'a. Znam najnowszą składnię ES6.

JavaScript

Skryptowy język programowania

Wiem jak za pomocą selektorów operować na dowolnym podzbiorze węzłów modelu DOM, przetwarzać im atrybuty. Manipulować modelem DOM oraz nadawać właściwości stylu elementom. Za pomocą jQuer potrafię obsługiwać zdarzenia oraz animację.

jQuery

Biblioteka JavaScript

Poznałem specyfikę działania motywów oraz wtyczek. Wiem jak trzorzyć motywy potomne (child theme) oraz własne szablony. Tworzyłem również niestandardowe typy postów.

WordPress

CMS System zarządzania treścią

Poznałem klasę "XMLHttpRequest" umożliwiająca asynchroniczne przesyłanie danych, bazującą na formacie wyniamy danych JSON

Ajax

Asynchronous JavaScript and XML

Potrafię wykorzystywać elementy Frameworku Bootstrap – tj. Grid RWD, elementy typografii, menu nawigacyjne, przyciski. Znam i stosuję regułę "mobile-first"

Bootstrap

Framework Bootstrap

Potrafię efektywnie korzystać z udogodnień dla programisty jakie oferuje SASS, począwszy od struktur zagnieżdżonych, poprzez mixiny, zmienne, kończąc na pętlach zarówno warunkowych jak i iteracyjnych. Pracując z SASS trzymam się zasady DRY :)

Sass
